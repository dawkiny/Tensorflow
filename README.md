## Install `cuda`

### Download `cuda runfile`

Download it [here](https://developer.nvidia.com/cuda-downloads)

### Exit `X server` if it is running

```sh
sudo service lightdm stop
```

```sh
sudo sh cuda_8.0.61_375.26_linux.run
```
